# 苏州骑行路线可视化

从相城区紫珺兰园出发的8条骑行路线可视化网页，支持查看详细路线图和一键导航。

## 功能特性

- 🚴 **8条精选路线**：从20km到280km，覆盖休闲到极限挑战
- 🗺️ **交互式地图**：Canvas绘制，无需外部依赖，离线可用
- 🧭 **骑行导航**：一键跳转高德/百度地图，自动设置途经点
- 📋 **途经点复制**：点击复制地名，方便手动设置导航
- 📱 **响应式设计**：适配手机和电脑
- 🌙 **深色主题**：护眼舒适

## 自定义修改

### 添加新路线
在 `index.html` 中找到 `const routes = [...]` 数组，按以下格式添加：

```javascript
{
  id: 9,                          // 唯一ID
  distance: 120,                  // 往返公里数
  name: '新路线名称',              // 路线名称
  level: 'hard',                  // 难度: easy/medium/hard/extreme
  badge: '进阶挑战',               // 标签
  path: '起点 → 途经点 → 终点',    // 简要路线
  duration: '6-7小时',            // 预计时间
  terrain: '平坦',                // 路面状况
  waypoints: [                    // 途经点坐标（GCJ-02坐标系）
    { name: '起点', desc: '描述', lat: 31.412, lng: 120.568 },
    // ... 更多途经点
  ],
  tips: ['建议1', '建议2'],       // 骑行建议
  color: '#a78bfa'                // 路线颜色
}
```

### 修改起点
搜索 `紫珺兰园` 替换为新起点名称，并更新对应的经纬度。

### 修改样式
CSS变量位于文件开头的 `:root` 中，可以修改：
- `--bg-primary`: 主背景色
- `--accent-teal`: 主强调色
- 其他颜色变量

## 技术栈

- **HTML5 + CSS3 + Vanilla JavaScript**
- **Canvas 2D**：地图绘制
- **Google Fonts**：Noto Sans SC 字体
- **高德地图 URI API**：导航跳转

## 坐标系说明

本项目使用 **GCJ-02（火星坐标系）**，这是高德地图和腾讯地图使用的坐标系。
如果从其他来源获取GPS坐标（WGS-84），需要先转换。
